## 🧠 What is this?
A hybrid quantum-classical model using IBM Qiskit to predict FIFA World Cup 2026 results. It analyzes 48 teams across 20 variables using a 6-qubit ZZFeatureMap circuit and runs 1000 Monte Carlo tournament simulations.

## 📊 20 Variables
Elo Rating, xG, xGA, FIFA Rank, Key Injuries, Squad Value, Average Age, Home Advantage, Temperature, Humidity, Rest Days, Form (5 & 10 games), H2H History, Striker Available, Goalkeeper Available, Goal Difference, Betting Odds, WC Experience, Tournament Phase.

## ⚡ Tech Stack
Qiskit · ZZFeatureMap · RealAmplitudes · Qiskit Aer · qiskit-machine-learning · pandas · numpy · matplotlib
